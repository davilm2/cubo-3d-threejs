# Cubo 3D com Three.js

## Descrição
Este projeto apresenta uma aplicação web desenvolvida com Three.js para renderizar um cubo em um ambiente 3D. A cena utiliza câmera em perspectiva e permite interação com o mouse para rotação, zoom e movimentação da câmera.

## Funcionalidades
- Exibição de um cubo 3D
- Câmera em perspectiva
- Rotação da câmera com o mouse
- Zoom com o scroll
- Movimentação da câmera
- Ajuste automático ao redimensionar a janela

## Tecnologias utilizadas
- HTML
- CSS
- JavaScript
- Three.js

## Como executar
1. Baixe ou clone este repositório
2. Abra o arquivo `index.html` no navegador

## Controles
- Botão esquerdo do mouse: rotacionar
- Scroll: zoom
- Botão direito do mouse: mover a câmera
